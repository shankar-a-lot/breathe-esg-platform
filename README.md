# ESG Emission Management Platform

AI-powered ESG data ingestion and emission tracking platform.

## Tech Stack
- Django REST Framework
- React.js
- PostgreSQL
- Render
- Vercel

## Features
- CSV Upload
- Emission Normalization
- ESG Dashboard
- Review Workflow
- REST APIs

## Run Backend
pip install -r requirements.txt
python manage.py runserver

## Run Frontend
cd frontend
npm install
npm run dev
